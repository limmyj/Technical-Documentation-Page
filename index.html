Introduction

Python is a popular programming language known for its simplicity and versatility. It is widely used in web development, data science, automation, and more.

Python's syntax is designed to be easy to read and understand, which makes it an excellent choice for beginners and professionals alike.

  • Python was created by Guido van Rossum and first released in 1991.
  • It is an interpreted, high-level, and general-purpose programming language.
  • Python emphasizes code readability with the use of significant whitespace.
Syntax Basics

Python syntax is designed to be intuitive and its code easy to read. Below is an example of a simple Python script:

print("Hello, World!")

Indentation is used to define code blocks in Python. For example:

if 5 > 2: print("Five is greater than two")

Python does not use curly braces ({}) to define code blocks like many other languages. Instead, it relies on indentation levels to determine the block structure.

Data Types

Python supports various data types, including:

  • Numbers: int, float, complex
  • Strings: str
  • Boolean: bool
  • Lists: list
  • Dictionaries: dict
  • Tuples: tuple

Below is an example of defining different data types in Python:

my_string = "Hello" my_int = 10 my_float = 20.5 my_list = [1, 2, 3]

Python's dynamic typing allows you to change the data type of a variable as the program runs.

Control Structures

Python includes several control structures for decision-making and looping, such as:

  • If statements
  • For loops
  • While loops

Example of an if-else statement in Python:

age = 18 if age >= 18: print("You are an adult.") else: print("You are still young.")
Functions

Functions in Python are defined using the def keyword. Functions can take arguments and return values.

Here's an example of a simple Python function:

def greet(name): return f"Hello, {name}!" print(greet("Alice"))

Functions help organize code into reusable blocks and improve code readability.
